# Desafio Tunts

## Instruções para utilizar o projeto:

* Clone o repositório

      git clone git@github.com:guilhermecd15/Tunts.git

* Entre na pasta do repositório que você acabou de clonar:

      cd Tunts

* Instale as dependências:

      npm install

* Insira a situação de cana aluno

      npm start
